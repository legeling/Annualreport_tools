# 需求文档

## 产品定位

财经报告小工具是一个面向本地使用的桌面应用，用于管理已有年报链接表，快速筛选、打开、下载、预览、转换和分析年报。应用不提供高并发爬虫能力，不鼓励批量高频访问数据源。

## 目标用户

- 已经通过脚本或数据集获得年报链接表的研究者、学生和内容生产者。
- 需要按公司、年份、行业快速定位年报的人。
- 需要把 PDF 年报转成文本并做关键词统计的人。

## MVP 范围

### 必须支持

1. 导入年报链接 Excel
   - 支持选择本地 Excel 文件。
   - 读取公司代码、公司简称、标题、年份、年报链接等字段。
   - 导入后在本地建立文档库索引。

2. 年报筛选与检索
   - 支持关键词搜索：公司名、股票代码、标题。
   - 支持年份、板块、行业、处理状态筛选。
   - 支持查看记录总数、已下载数、已转换数、已分析数。

3. 年报打开与下载
   - 未下载记录可以打开远程链接。
   - 支持单个 PDF 下载到本地目录。
   - 支持低并发下载队列，并提供失败重试。

4. PDF 预览
   - 已下载 PDF 可在本地打开。
   - 第一版允许调用系统默认 PDF 阅读器。
   - 后续版本支持应用内嵌预览。

5. PDF 转文本
   - 支持将本地 PDF 转换成 TXT。
   - 记录转换状态和输出路径。

6. 文本分析
   - 支持关键词词频统计。
   - 支持导出分析结果 Excel。

### 明确不做

- 不做高并发爬虫。
- 不做账号系统。
- 不做云端同步。
- 不做在线协作。
- 不在 MVP 中实现复杂数据库服务。

## 核心流程

1. 用户导入年报链接表。
2. 应用解析并显示年报列表。
3. 用户通过搜索和筛选定位目标年报。
4. 用户打开远程链接或下载 PDF。
5. 用户预览本地 PDF。
6. 用户将 PDF 转成文本。
7. 用户运行关键词分析并导出结果。

## 数据字段

基础字段：

- company_code：公司代码
- company_name：公司简称
- title：公告标题
- year：年报年份
- report_url：年报链接
- market：板块
- industry：行业
- local_pdf_path：本地 PDF 路径
- local_txt_path：本地 TXT 路径
- status：处理状态
- updated_at：更新时间

## 处理状态

- 未下载
- 已下载
- 已转文本
- 已分析
- 下载失败
- 转换失败
- 分析失败

## 成功标准

- 用户能导入一份年报链接表并完成本地检索。
- 用户能单击打开远程链接或本地 PDF。
- 用户能下载单份 PDF。
- 用户能对已下载 PDF 执行转文本。
- 用户能对文本运行关键词分析并导出结果。
